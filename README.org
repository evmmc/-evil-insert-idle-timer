#+title: Evil Insert Idle Timer Mode

When in /insert/ mode of [[https://www.emacswiki.org/emacs/Evil][EmacsWiki: Evil]] , switch to /normal/ mode after N seconds of idleness.

* Usage

#+begin_src emacs-lisp :tangle yes
(use-package! +evil-insert-idle-timer
  :after evil
  :config
  (setq +evil-insert-idle-timer-delay 4)
  (+evil-insert-idle-timer-mode 1))
#+end_src
